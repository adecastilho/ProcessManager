How to compile and run:
* Open cmd terminal and navigate to the folder containing the unzipped project files
* Run the Makefile with cmd 'make all'
* Run PMan with cmd './PMan'

Commands accepted by PMan:
'bg <program path and optional flags>'
'bglist'
'bgkill <pid>'
'bgstop <pid>'
'bgstart <pid>'
'pstat <pid>'

See project spec for more details on PMan usage.
